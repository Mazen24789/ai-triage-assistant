# 🚨 AI Emergency Triage Assistant

This project predicts patient urgency (CTAS level) using vital signs like heart rate, temperature, and oxygen saturation.

## Features
- Real-time prediction
- Risk classification (high, medium, low)
- Emergency alerts

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## How to Run

pip install streamlit scikit-learn numpy
python3 -m streamlit run app.py
